Account Settings

Add your account number JSON, with color modifications if required. HTTPS/HTTP entries will add items to the drop-down menu.

Example 

          {
            urls: [
              {
                "title": "Google",
                "url": "https://google.com",
                "id": "0",
                "groupId": 0
              },
              {
                "title": "Microsoft",
                "url": "https://microsoft.com",
                "id": "1",
                "groupId": 0
              }
            ],
            groups: [
              {
                "id": 0,
                "title": "Default",
                "sortUrlsSwitch": "true",
                "useContainerSwitch": "false"
              },
            ],
            version: "5.5.5.1",
            preferences: {
              awsConsole: {
                compressMode: false
              }
            }
          }
